pub struct Interface { /* fields omitted */ }

Representation of an interface in a grammar declaration.

Interfaces represent nodes in the AST. Each interface has a name, a type, defines properties (also known as attribute in webidl) which hold values. Interfaces may also have meta-properties, such as their skippability.
