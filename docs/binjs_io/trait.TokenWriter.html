Build an in-memory representation of a BinTree.

Implementations may for instance introduce atoms, maximal sharing, etc.

Associated Types

type Tree

The type of trees manipulated by this writer.

type Statistics

Statistics produced by this writer.

type Data: AsRef<[u8]>

The type of data generated by this writer. Typically some variant of Vec<u8>.

type Error

An error returned by this writer.

Note that errors are not recoverable within the life of this TokenWriter.

Write a tagged tuple.

fn list(&mut self, _: Vec<Self::Tree>) -> Result<Self::Tree, Self::Error>

Write a list.
